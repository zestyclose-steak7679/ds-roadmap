Connect Firebase Cloud Sync
Follow these 4 steps to link your free Firebase database. Your progress will sync across every device in real time. Takes about 5 minutes.
1
Create a Firebase project
Go to console.firebase.google.com and sign in with your Google account.

Click "Create a project" → Enter a name like ds-roadmap → Click Continue → Disable Google Analytics (not needed) → Click "Create project" → Continue.
Firebase Console ┌─────────────────────────────────────┐ │ + Create a project │ │ Name: ds-roadmap │ │ Analytics: OFF ← disable this │ │ │ │ [ Create project ] │ └─────────────────────────────────────┘
2
Enable Realtime Database
Inside your project, click Build → Realtime Database in the left sidebar.

Click "Create Database" → Choose location: United States (us-central1) → Click Next → Select "Start in test mode" → Click Enable.
Realtime Database setup ┌─────────────────────────────────────┐ │ Build → Realtime Database │ │ → Create Database │ │ → Location: us-central1 │ │ → Start in test mode │ ← important │ → [ Enable ] │ └─────────────────────────────────────┘
Then click the Rules tab and replace the content with this — it allows read/write for 10 years:
{ "rules": { ".read": "now < 1893456000000", ".write": "now < 1893456000000" } }
Click Publish after pasting the rules.
3
Register a Web App & get your config
Go back to Project Overview (click the ⌂ home icon top-left).

Click the </> Web icon → Enter app nickname: ds-roadmap-web → Click "Register app".

You'll see a config block like this. Copy each value and paste it below:
Your Firebase config (example — yours will be different) const firebaseConfig = { apiKey: "AIzaSyXXXXXXXXXXXXXXXXXXXXXX", authDomain: "ds-roadmap.firebaseapp.com", databaseURL: "https://ds-roadmap-default-rtdb.firebaseio.com", projectId: "ds-roadmap", storageBucket: "ds-roadmap.appspot.com", messagingSenderId: "123456789012", appId: "1:123456789012:web:abcdefabcdef" };
apiKey
authDomain
databaseURL
projectId
storageBucket
messagingSenderId
appId
These keys are safe to use in a frontend app — Firebase security is handled by your database rules above.
4
Test connection & save
Click the button below — it will test the connection to your Firebase database and save the config permanently in this browser.
