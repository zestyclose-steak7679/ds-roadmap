Deploy to GitHub Pages free · permanent URL
Follow these steps once. You'll get a live URL like chinmay.github.io/ds-roadmap that works on any device.
1
Download this file
Click "Download HTML" button in the header. Save it as index.html
2
Create a new GitHub repository
Go to github.com/new → Name it ds-roadmap → Set to Public → Click "Create repository"
3
Upload the file
On your new repo page → click "uploading an existing file" → drag and drop index.html → click "Commit changes"
4
Enable GitHub Pages
Go to repo → Settings → Pages (left sidebar) → Source: "Deploy from a branch" → Branch: main → folder: / (root) → Save
5
Your site is live!
Wait ~60 seconds → visit https://YOUR-USERNAME.github.io/ds-roadmap — bookmark it on every device.
!
Note on cross-device progress
Progress saves to localStorage in each browser. To share progress between devices, use the same browser signed into the same profile (e.g. Chrome sync), or export/import using the buttons in the header.
